# Project T: Real vs. Fake Face Classification

Run order:

1. Open `Project_T_Train.ipynb`.
2. Use the `python3.11` Jupyter kernel. The notebooks are already set to that kernel.
3. Install the packages in `requirements.txt` if needed.
4. Run all training notebook cells to download the Kaggle data, train/validate/test ResNet-18, and save `resnet18_fake_faces.pt`.
5. Open `Project_T_Demo.ipynb`.
6. Run all demo notebook cells to load the saved model and evaluate the 10 independent images in `10_Custom_Test_Images`.

The demo notebook does not include training code, matching the Project T handout requirement.
