# Custom Test Images

These are the 10 independent examples used by `Project_T_Demo.ipynb`.

The five fake examples were collected from ThisPersonDoesNotExist.com by refreshing the page and saving generated face images. The five real examples were collected from Wikimedia Commons public image files using the file names listed in `custom_test_labels.csv`.

These examples are separate from the Kaggle training/validation/test dataset used in `Project_T_Train.ipynb`.
